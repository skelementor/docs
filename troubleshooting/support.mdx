## Before Contacting Support

### Check Documentation

- Review the [troubleshooting guide](/troubleshooting/common-errors)
- Browse the [FAQ](/troubleshooting/faq)
- Check [installation guide](/installation)

### Gather Information

When contacting support, please include:

- **WordPress version**
- **Elementor version**
- **PHP version**
- **Plugin version**
- **Error messages** (if any)
- **Steps to reproduce** the issue
- **Screenshots** (if helpful)

### Try Basic Troubleshooting

<Steps>
  <Step title="Clear Caches">Clear Elementor cache and browser cache</Step>
  <Step title="Check Requirements">
    Verify WordPress, Elementor, and PHP versions
  </Step>
  <Step title="Test with Default Theme">
    Switch to a default WordPress theme temporarily
  </Step>
  <Step title="Deactivate Other Plugins">
    Test with only Skelementor active
  </Step>
</Steps>

## Feature Requests

### Suggest New Features

- **GitHub Issues:** Create a feature request issue
- **Email:** Send detailed feature requests to support
- **Community:** Discuss ideas in WordPress.org forums

### What to Include

- **Use case:** Why do you need this feature?
- **Examples:** How would it work?
- **Priority:** How important is this to you?

## Bug Reports

### Report Bugs

- **GitHub Issues:** Create a bug report issue
- **Email:** Send detailed bug reports to support

### What to Include

- **Steps to reproduce:** Exact steps to trigger the bug
- **Expected behavior:** What should happen?
- **Actual behavior:** What actually happens?
- **Environment:** WordPress, Elementor, PHP versions
- **Error messages:** Any error messages or logs
