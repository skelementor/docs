## Display

Control how elements are displayed with our display utilities.

<Card title="Display Properties" icon="grid">

|                 |              |                          |                              |
| --------------- | ------------ | ------------------------ | ---------------------------- |
| `.block`        | Block        | `display: block;`        | Full-width block element     |
| `.inline-block` | Inline-block | `display: inline-block;` | Inline with block properties |
| `.flex`         | Flex         | `display: flex;`         | Flexbox container            |
| `.inline-flex`  | Inline-flex  | `display: inline-flex;`  | Inline flexbox container     |
| `.hidden`       | Hidden       | `display: none;`         | Hide element                 |

</Card>

### Examples

```html
<!-- Display examples -->
<div class="block p-4 bg-gray-light">Block element</div>
<span class="inline-block p-2 bg-blue text-white">Inline block</span>
<div class="flex items-center gap-4">Flex container</div>
<div class="inline-flex items-center gap-2">Inline flex</div>
<div class="hidden">This is hidden</div>
```

## Overflow

Control how content overflows its container.

<Card title="Overflow Properties" icon="arrows-left-right">

|                    |        |                     |                             |
| ------------------ | ------ | ------------------- | --------------------------- |
| `.overflow-hidden` | Hidden | `overflow: hidden;` | Hide overflowing content    |
| `.overflow-auto`   | Auto   | `overflow: auto;`   | Show scrollbars when needed |

</Card>

### Examples

```html
<!-- Overflow examples -->
<div class="w-64 h-32 overflow-hidden border border-gray">
  <p class="p-4">
    This content might overflow the container and will be hidden.
  </p>
</div>

<div class="w-64 h-32 overflow-auto border border-gray">
  <p class="p-4">This content will show scrollbars when it overflows.</p>
</div>
```

## Object Fit

Control how images and videos fit within their containers.

<Card title="Object Fit Properties" icon="image">

|                   |         |                        |                                   |
| ----------------- | ------- | ---------------------- | --------------------------------- |
| `.object-cover`   | Cover   | `object-fit: cover;`   | Cover container, crop if needed   |
| `.object-contain` | Contain | `object-fit: contain;` | Fit within container, no cropping |

</Card>

### Examples

```html
<!-- Object fit examples -->
<img src="image.jpg" class="w-64 h-32 object-cover rounded" alt="Cover image" />
<img
  src="image.jpg"
  class="w-64 h-32 object-contain border"
  alt="Contain image"
/>
```

## Aspect Ratio

Control the aspect ratio of elements.

<Card title="Aspect Ratio Properties" icon="square">

|                  |      |                         |                     |
| ---------------- | ---- | ----------------------- | ------------------- |
| `.aspect-square` | 1:1  | `aspect-ratio: 1 / 1;`  | Square aspect ratio |

</Card>

### Examples


## Flexbox Direction

Control the direction of flex items.

<Card title="Flex Direction" icon="square-right">

|             |        |                           |                             |
| ----------- | ------ | ------------------------- | --------------------------- |
| `.flex-row` | Row    | `flex-direction: row;`    | Horizontal layout (default) |
| `.flex-col` | Column | `flex-direction: column;` | Vertical layout             |

</Card>

### Examples

```html
<!-- Flex direction examples -->
<div class="flex flex-row gap-4">
  <div class="p-4 bg-blue text-white">Item 1</div>
  <div class="p-4 bg-green text-white">Item 2</div>
  <div class="p-4 bg-red text-white">Item 3</div>
</div>

<div class="flex flex-col gap-4">
  <div class="p-4 bg-blue text-white">Item 1</div>
  <div class="p-4 bg-green text-white">Item 2</div>
  <div class="p-4 bg-red text-white">Item 3</div>
</div>
```

## Justify Content

Control horizontal alignment of flex items.

<Card title="Justify Content" icon="square-chevron-right">

|                    |               |                                   |                     |
| ------------------ | ------------- | --------------------------------- | ------------------- |
| `.justify-start`   | Start         | `justify-content: start;`         | Align to start      |
| `.justify-center`  | Center        | `justify-content: center;`        | Center alignment    |
| `.justify-end`     | End           | `justify-content: end;`           | Align to end        |
| `.justify-between` | Space between | `justify-content: space-between;` | Space between items |

</Card>

### Examples

```html
<!-- Justify content examples -->
<div class="flex justify-start gap-4 p-4 bg-gray-light">
  <div class="p-2 bg-blue text-white">Start</div>
  <div class="p-2 bg-green text-white">Items</div>
</div>

<div class="flex justify-center gap-4 p-4 bg-gray-light">
  <div class="p-2 bg-blue text-white">Centered</div>
  <div class="p-2 bg-green text-white">Items</div>
</div>

<div class="flex justify-between p-4 bg-gray-light">
  <div class="p-2 bg-blue text-white">Start</div>
  <div class="p-2 bg-green text-white">End</div>
</div>
```

## Align Self

Control individual item alignment within a flex container.

<Card title="Align Self" icon="user">

|                |        |                       |                  |
| -------------- | ------ | --------------------- | ---------------- |
| `.self-center` | Center | `align-self: center;` | Center this item |
| `.self-start`  | Start  | `align-self: start;`  | Align to start   |
| `.self-end`    | End    | `align-self: end;`    | Align to end     |

</Card>

### Examples

```html
<!-- Align self examples -->

  <div class="p-2 bg-red text-white">End</div>
</div>
```

## Gap

Control spacing between flex items.

<Card title="Gap Properties" icon="down-left-and-up-right-to-center">

|          |      |              |                 |
| -------- | ---- | ------------ | --------------- |
| `.gap-1` | 4px  | `gap: 4px;`  | Small gap       |
| `.gap-2` | 8px  | `gap: 8px;`  | Small gap       |
| `.gap-3` | 12px | `gap: 12px;` | Medium gap      |
| `.gap-4` | 16px | `gap: 16px;` | Standard gap    |
| `.gap-6` | 24px | `gap: 24px;` | Large gap       |
| `.gap-8` | 32px | `gap: 32px;` | Extra large gap |

</Card>

### Examples

```html
<!-- Gap examples -->
<div class="flex gap-2">
  <div class="p-2 bg-blue text-white">Item 1</div>
  <div class="p-2 bg-green text-white">Item 2</div>
  <div class="p-2 bg-red text-white">Item 3</div>
</div>

<div class="flex gap-6">
  <div class="p-4 bg-blue text-white">Item 1</div>
  <div class="p-4 bg-green text-white">Item 2</div>
  <div class="p-4 bg-red text-white">Item 3</div>
</div>
```

## Best Practices

### Flexbox Layouts

<Steps>
  <Step title="Use Semantic HTML">
    Always use proper HTML elements with flexbox utilities.
  </Step>
  <Step title="Consistent Gaps">
    Use the gap utility instead of margin for spacing between flex items.
  </Step>
  <Step title="Responsive Design">
    Consider how layouts will work on different screen sizes.
  </Step>
  <Step title="Accessibility">
    Ensure flexbox layouts work with screen readers and keyboard navigation.
  </Step>
</Steps>

### Performance

- Flexbox is well-optimized in modern browsers
- Use `gap` instead of margin for better performance
- Avoid deeply nested flexbox containers when possible
