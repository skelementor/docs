## Color Accessibility

### Contrast Requirements

<Warning>
  **Accessibility Note:** Ensure sufficient color contrast for readability. The
  color palette is designed to meet WCAG AA standards.
</Warning>

<Card title="Contrast Guidelines" icon="eye">
  - **Normal text:** Minimum 4.5:1 contrast ratio - **Large text:** Minimum 3:1
  contrast ratio - **UI components:** Minimum 3:1 contrast ratio - **Focus
  indicators:** Minimum 3:1 contrast ratio
</Card>

### Color Blindness Considerations

<Card title="Color Blindness Support" icon="eye-slash">
  - Don't rely solely on color to convey information - Use icons, text, or
  patterns alongside color - Test with color blindness simulators - Provide
  alternative ways to identify status
</Card>

## Color Best Practices

### Semantic Usage

<Steps>
  <Step title="Use Semantic Colors">
    Use colors that match their semantic meaning (green for success, red for
    errors).
  </Step>
  <Step title="Maintain Consistency">
    Use the same colors for the same purposes throughout your design.
  </Step>
  <Step title="Consider Context">
    Choose colors that work well with your brand and content.
  </Step>
  <Step title="Test Accessibility">
    Ensure all color combinations meet accessibility standards.
  </Step>
</Steps>

### Color Hierarchy

<Card title="Color Hierarchy" icon="list">
  1. **Primary:** Blue for main actions and links 2. **Secondary:** Gray for
  supporting text and elements 3. **Success:** Green for positive feedback 4.
  **Warning:** Yellow for cautionary messages 5. **Error:** Red for errors and
  destructive actions 6. **Neutral:** White, black, and grays for structure
</Card>

## Custom Colors

### Adding Custom Colors

To add your own colors, create CSS with your custom color utilities:

```css
/* Custom brand colors */
.text-brand {
  color: #your-brand-color;
}
.bg-brand {
  background-color: #your-brand-color;
}
.border-brand {
  border-color: #your-brand-color;
}

/* Custom accent colors */
.text-accent {
  color: #your-accent-color;
}
.bg-accent {
  background-color: #your-accent-color;
}
```

### Color Variables

For better maintainability, consider using CSS custom properties:

```css
:root {
  --brand-primary: #your-brand-color;
  --brand-secondary: #your-secondary-color;
}

.text-brand {
  color: var(--brand-primary);
}
.bg-brand {
  background-color: var(--brand-primary);
}
```
