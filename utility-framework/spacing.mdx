## Spacing Best Practices

### Consistent Scale

<Steps>
  <Step title="Use the 4px Scale">
    Always use the predefined spacing scale (1, 2, 3, 4, 6, 8, 12) for
    consistency.
  </Step>
  <Step title="Maintain Vertical Rhythm">
    Use consistent spacing between related elements for better visual flow.
  </Step>
  <Step title="Group Related Elements">
    Use smaller gaps within groups and larger gaps between groups.
  </Step>
  <Step title="Consider Content Density">
    Adjust spacing based on content density and user experience needs.
  </Step>
</Steps>

### Responsive Spacing

```html
<!-- Responsive spacing example -->
<div class="p-4 md:p-6 lg:p-8">
  <h2 class="text-2xl md:text-3xl font-700 mb-4 md:mb-6">Title</h2>
  <p class="text-base md:text-lg leading-relaxed">
    Content with responsive spacing
  </p>
</div>
```

### Spacing Hierarchy

<Card title="Spacing Hierarchy" icon="list">
  1. **Micro spacing (1-2):** Fine adjustments, icon spacing 2. **Small spacing
  (3-4):** Element internal spacing, form fields 3. **Medium spacing (6):**
  Section internal spacing, card padding 4. **Large spacing (8-12):** Section
  spacing, major layout divisions
</Card>

## Common Mistakes

<Warning>**Avoid these spacing mistakes:**</Warning>

- Don't mix different spacing scales inconsistently
- Don't use margin for spacing between flex items (use gap instead)
- Don't forget to consider mobile spacing needs
- Don't overuse auto margins for centering (use flexbox utilities instead)
