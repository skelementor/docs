## Flexbox Direction

Control the direction of flex items.

<Card title="Flex Direction" icon="arrows-left-right">

|             |        |                           |                   |
| ----------- | ------ | ------------------------- | ----------------- |
| `.flex-row` | Row    | `flex-direction: row;`    | Horizontal layout |
| `.flex-col` | Column | `flex-direction: column;` | Vertical layout   |

</Card>

## Flexbox Wrap

Control how flex items wrap.

<Card title="Flex Wrap" icon="arrows-left-right-to-line">
  | | | | | 
  |------|------|-----|-------| 
  | `.flex-wrap` | Wrap | `flex-wrap: wrap;` | Allow wrapping | 
  | `.flex-nowrap` | No wrap | `flex-wrap: nowrap;` | Prevent wrapping |
</Card>

## Justify Content

Control horizontal alignment of flex items.

<Card title="Justify Content" icon="arrow-down-left-and-arrow-up-right-to-center">
  | | | | | 
  |------|---------|-----|-------| 
  | `.justify-start` | Start | `justify-content: start;` | Align to start | 
  | `.justify-center` | Center | `justify-content: center;` | Center alignment | 
  | `.justify-end` | End | `justify-content: end;` | Align to end | 
  | `.justify-between` | Space between | `justify-content: space-between;` | Space between items |
</Card>

## Align Items

Control vertical alignment of flex items.

<Card title="Align Items" icon="objects-align-left">
  | | | | | 
  |------|-------|-----|-------| 
  | `.items-start` | Start | `align-items: start;` | Align to start | 
  | `.items-center` | Center | `align-items: center;` | Center alignment | 
  | `.items-end` | End | `align-items: end;` | Align to end | 
  | `.items-stretch` | Stretch | `align-items: stretch;` | Stretch to fill |
</Card>

## Align Self

Control individual item alignment.

<Card title="Align Self" icon="user">
  | | | | | 
  |------|-------|-----|-------| 
  | `.self-center` | Center | `align-self: center;` | Center this item | 
  | `.self-start` | Start | `align-self: start;` | Align to start | 
  | `.self-end` | End | `align-self: end;` | Align to end |
</Card>

## Gap

Control spacing between flex items.

<Card title="Gap Properties" icon="arrows-left-right-to-line">
  | | | | | 
  |------|-----|-----|-------| 
  | `.gap-1` | 4px | `gap: 4px;` | Small gap | 
  | `.gap-2` | 8px | `gap: 8px;` | Small gap | 
  | `.gap-3` | 12px | `gap: 12px;` | Medium gap | 
  | `.gap-4` | 16px | `gap: 16px;` | Standard gap |  
  | `.gap-6` | 24px | `gap: 24px;` | Large gap | 
  | `.gap-8` | 32px | `gap: 32px;` | Extra large gap |
</Card>

## Common Flexbox Patterns

### Header Layout

```html
<header class="flex items-center justify-between p-6 bg-white">
  <h1 class="text-2xl font-700">Logo</h1>
  <nav class="flex gap-6">
    <a href="#" class="text-gray">Home</a>
    <a href="#" class="text-gray">About</a>
  </nav>
</header>
```

### Centered Content

```html
<div class="flex items-center justify-center min-h-screen">
  <div class="text-center">
    <h1 class="text-4xl font-700 mb-4">Welcome</h1>
    <button class="btn btn-blue">Get Started</button>
  </div>
</div>
```

### Card Grid

```html
<div class="flex flex-wrap gap-6">
  <div class="w-1-3">
    <div class="card p-6">Card 1</div>
  </div>
  <div class="w-1-3">
    <div class="card p-6">Card 2</div>
  </div>
  <div class="w-1-3">
    <div class="card p-6">Card 3</div>
  </div>
</div>
```
